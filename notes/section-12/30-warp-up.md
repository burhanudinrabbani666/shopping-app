# Module Summary

## NoSQL / MongoDB

- Alternative to SQL Database
- No strict schemas, fewer realtions
- You can of course use schemas and reference-based relations but you gor more flexibility
- Often, relations are also creted sby embedding other documents/ data

## Working with mongo BD

- Use the officil mongoDB Driver
- Commands like insert one(), find(), updateOne(), and deleteOne() make CRUD operations very simple
- Check the offical docs to learn about all avaible operations + configurations / operators
- All operations are promise-based, hence you can easly chain them for more complex flows

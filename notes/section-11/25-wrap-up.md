# Warp up

## SQL

- SQL is strict data schemas and relations
- You can connect your node.js app via packages like `mysql` or `pg`
- Writing SQL queries is not directly related to node.js and something you have to learn to node.js

## Sequelize

- Instead of writing SQL queries manually, you can use packages (ORMs) like sequlize to focus on the node.js code and work with native JS object
- Sequlize allows you define models nad interact with the database through them
- You can also easly set up relations ("Associations") and interact with your related models through them
